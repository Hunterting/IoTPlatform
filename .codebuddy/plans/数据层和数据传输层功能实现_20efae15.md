## 产品概述

IoT平台后端API系统的数据层和数据传输层实现，基于.NET 8构建的物联网SaaS平台后端API系统。

## 核心功能

- 数据层仓储模式实现：提供通用的CRUD操作和数据访问抽象
- 数据传输层映射配置：实现实体模型到DTO的自动转换
- 初始化数据脚本：创建开发和测试所需的种子数据
- 多租户数据隔离支持：通过AppCode字段实现数据隔离
- 区域权限过滤支持：通过AllowedAreaIds字段实现权限控制

## 技术栈选择

- **后端框架**: .NET 8 Web API (与现有项目保持一致)
- **ORM**: Entity Framework Core 8.0 (与现有项目保持一致)
- **数据库**: MySQL 5.7 (Pomelo.EntityFrameworkCore.MySql 8.0.2)
- **对象映射**: AutoMapper 12.0.1 (已安装)
- **设计模式**: 仓储模式 + 单元工作模式

## 实现方案

### 高层面策略

采用标准的仓储模式设计，提供通用的CRUD操作接口，支持多租户数据隔离和区域权限过滤。实现AutoMapper映射配置，简化实体到DTO的转换。创建种子数据脚本，用于系统初始化。

### 关键技术决策

1. **仓储模式设计**：

- 采用泛型仓储接口`IRepository<T>`，提供基本CRUD操作
- 实现泛型仓储基类`Repository<T>`，封装EF Core数据访问逻辑
- 为每个实体创建特定仓储接口（如`IUserRepository`），扩展特定业务操作
- 实现单元工作模式`IUnitOfWork`，支持事务管理

2. **AutoMapper配置**：

- 创建`MappingProfile`类，配置所有实体到DTO的映射规则
- 支持嵌套对象的映射配置
- 支持自定义映射转换器

3. **多租户支持**：

- 所有仓储操作自动应用`AppCode`过滤
- 支持超级管理员查看所有租户数据
- 通过`TenantFilterAttribute`实现数据隔离

4. **性能考虑**：

- 使用EF Core的Include()预加载关联数据，避免N+1查询
- 实现查询缓存机制（可选）
- 使用异步操作提高并发处理能力

### 架构设计

#### 系统架构

采用分层架构模式，数据层作为数据访问层（DAL），为业务服务层提供数据访问抽象。

```
┌─────────────────────────────────────────────┐
│            业务服务层 (Services)            │
├─────────────────────────────────────────────┤
│         数据访问层 (Repositories)           │
├─────────────────────────────────────────────┤
│      实体框架层 (Entity Framework)         │
├─────────────────────────────────────────────┤
│           数据库层 (MySQL)                 │
└─────────────────────────────────────────────┘
```

#### 数据访问模式

```
Controller → Service → Repository → DbContext → Database
    ↓          ↓          ↓           ↓           ↓
   DTOs      DTOs      Entities   Entities     Tables
    ↑          ↑          ↑           ↑           ↑
  映射配置    映射配置    映射配置    映射配置    映射配置
```

### 关键代码结构

```
// 泛型仓储接口
public interface IRepository<T> where T : class
{
    Task<T?> GetByIdAsync(long id);
    Task<IEnumerable<T>> GetAllAsync();
    Task<PagedResult<T>> GetPagedAsync(int page, int pageSize);
    Task<T> AddAsync(T entity);
    Task UpdateAsync(T entity);
    Task DeleteAsync(T entity);
    IQueryable<T> GetQueryable();
}

// 单元工作接口
public interface IUnitOfWork : IDisposable
{
    IRepository<T> GetRepository<T>() where T : class;
    Task<int> SaveChangesAsync();
    Task BeginTransactionAsync();
    Task CommitTransactionAsync();
    Task RollbackTransactionAsync();
}

// AutoMapper映射配置
public class MappingProfile : Profile
{
    public MappingProfile()
    {
        CreateMap<User, UserDto>();
        CreateMap<CreateUserRequest, User>();
        CreateMap<UpdateUserRequest, User>();
        // ... 其他映射配置
    }
}
```
